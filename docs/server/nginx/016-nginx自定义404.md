# 016-nginx自定义404

## 1、nginx配置
nginx自带404页面，当我们想要定制的时候，只需要配置`error_page  404`属性即可
```nginx
server {
    listen       80;
    server_name  aaa.com www.aaa.com;

    location / {
        root   /root/svr/aaa;
        index  index.html index.htm;
    }
    
    # 自定义404页面
    error_page  404 /404.html;
}
```

上面当用户访问`http://aaa.com/sdfjsdfj234`一个不存在的页面的时候，就会展示我们的404页面，并且http状态是404
