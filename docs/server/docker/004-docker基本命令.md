# 004-docker基本命令

docker的基本命令，控制docker本身进程


## 1、启动
```shell
systemctl start docker
```

## 2、停止
```shell
systemctl stop docker
```

## 3、重启
```shell
systemctl restart docker
```

## 4、开机自启动
```shell
systemctl enable docker
```

## 5、查看概要
```shell
docker info 
```

## 6、查看帮助文档
```shell
docker --help
```

## 6、查看版本号
```shell
docker version
```
