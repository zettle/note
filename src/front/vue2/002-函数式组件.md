# 002-函数式组件


## 1、react的函数组件
```jsx
function App () {
  return <h1>Hello</h1>
}
```


## 2、vue的函数组件

在`<template>`上加上`functional`关键词表示这是一个函数组件

vue的函数组件就不能再写`<script>`标签了

```vue
<template functional>
    <div>{{props.title}}</div>
</template>

<style>
</style>
```

总的说，函数组件具有以下特点:

* 无状态，无法实例，没有this，所以比如要用emit和父组件通讯都不行
* 只接受props
* 性能好
* 无`<script>`标签，即无法写业务逻辑

这些特点和react的函数组件还是很类似的
