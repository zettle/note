# 002-处理css

vite自带postcss，所以直接在根目录新建`postcss.config.js`即可配置


## 1、px2rem
安装: `npm i -D postcss-pxtorem`

配置:
```js
module.exports = {
    'plugins': {
        'postcss-pxtorem': {
            rootValue: 75,
            propList: ['*'],
            minPixelValue: 2, // 小于2px的不会转为rem，等于2的还是会转
            selectorBlackList: [/^.px-\w*/], // 不要转vant的，他们是375位基准的
            exclude: /node_modules/i // 忽略node_modules里面的
        }
    }
}
```
